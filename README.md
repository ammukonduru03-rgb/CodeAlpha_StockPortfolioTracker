# 📈 CodeAlpha - Stock Portfolio Tracker

## 📌 Project Overview

The Stock Portfolio Tracker is a Python-based console application developed as part of the CodeAlpha Python Programming Internship.

The program allows users to enter stock names and quantities, calculates the total investment, validates user input, and saves the investment summary to a text file.

---

## ✨ Features

- Add multiple stock investments
- Calculate total investment value
- Validate stock names
- Supports uppercase and lowercase user input
- Save investment summary to a text file
- Beginner-friendly Python implementation

---

## 🛠️ Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements
- File Handling

---

---

## ▶️ How to Run

1. Clone this repository.
2. Open `stock_portfolio_tracker.py`.
3. Run the program.
4. Enter stock names and quantities.
5. Type `no` when finished.
6. View the total investment on the console.
7. Check `investment.txt` for the saved summary.

---
